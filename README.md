# Luxe Dental Studio

Hero cinematográfico com scroll storytelling para clínica odontológica premium.

## Deploy

```bash
# Vercel
npx vercel --prod

# Ou arraste a pasta para vercel.com
```

## Estrutura

```
index.html     ← Site completo (hero com scroll animation)
assets/        ← Imagens originais e otimizadas
vercel.json    ← Config de deploy
```
